  • 在导航栏添加“关于”页面

    通过简单的几个命令，即可在导航栏中添加“关于页面”，这个思路也可用于添加其它页面。具体步骤如下：

    1. 在命令行运行命令：hexo new page "about"
    2. 在source/about/index.md中编辑页面内容

    3. 在themes/你的主题/_config.yml中添加如下内容：

      menu:
    关于: about

  • 在正文中添加代码块

    在需要添加代码块的地方可以使用markdown语法，即在代码前面留四个空格，或者使用一对’`’将代码部分括起来，也可以使用hexo帮助文档中的语法结构Tag Plugins。

  • 在使用hexo new命令新建博文时自动生成Post Asset Folder

    如果博文中经常需要引用图片等多媒体资源，通过更改全局设置，可以使得运行hexo new命令新建博文时自动在source/_post文件夹下生成同名的资源文件夹，所需图片等资源可以放在该文件夹下便于引用。具体做法如下：

    在全局设置文件_config.yml（不是模板文件下的同名文件）中添加如下内容：

    post_asset_folder: true

    在博文中引用资源文件的命令可参照hexo官方文档asset folders。
