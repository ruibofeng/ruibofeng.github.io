Hexo简明教程（二）

将博客托管到Github

托管博客到Github是利用Github提供的GithubPages功能，具体步骤可参考前面提到的网页，现简述如下：

  1. 在Github中注册一个账号。

  2. 在Github上创建一个仓库，将其命名为username.github.io，这里的username指的是你在github上的用户名，这里千万不要改为其它名字。

  3. 修改hexo本地配置文件（根目录下的_config.yml，而非themes目录下的_config.yml），具体内容如下：

    需要注意的是，上面的仓库地址需要根据个人情况设置，这里采用的是git协议，采用https协议也可以，具体操作时选取任意一个能够使用的即可。

  4. 部署本地博客到GitHub：

    部署之前，需要安装一个插件，执行以下命令完成安装：

    npm install hexo-deployer-git --save

    插件完成之后，执行以下命令即可完成部署：

    hexo new article_title #新建一篇文章
hexo generate #生成静态网页
hexo deploy #部署到Github

进行完上面的设置之后，在浏览器访问username.github.io（需将username替换成你自己在github的用户名）就可以打开托管在Github上的博客了。

绑定独立域名

    域名服务商很多，这里推荐使用godaddy，具体流程在官网上很容易实现，网上各种教程也不少，这里不再赘述。
